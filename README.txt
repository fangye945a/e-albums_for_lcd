							 多功能电子相册

一、本项目基于粤嵌学习第一阶段项目所需而编写。

二、本项目为一个基于粤嵌科技arm210开发板的多功能电子相册，可读取像素为（长）800*（宽）480的bmp和jpg格式图片。

三、本项目主要功能及用法：
	
	运行程序，读出当前目录jpg和bmp格式文件，并将文件名和总数量列出。打开当前目录下第一张图片，然后进入等待命令模式。
	
	等待命令模式下从键盘输入指令则执行对应动作。

	1.指令 n  查看下一张图片   

	2.指令 N  查看上一张图片

	3.指令 i  指定区域画颜色块 
		
		需要从键盘输入x_Start（x轴起始坐标）、y_Start（y轴起始坐标）、x_End（x轴结束坐标）、 y_End （y轴起始坐标、color（块颜色）shape（块形状）
		
	形状表：
		形状：     值
		矩形        0
 		圆形        1
	
	颜色表如下图所示：

		  颜色：    值			
		//蓝色		0
		//紫色 		1
		//绿色		2
		//红色		3
		//黄色		4
		//天蓝色	5
		//白色		6
		//黑色	
	色块添加完成后，回到等待输入命令模式。	

	4.指令 q  退出当前程序
	
	5.指令 t  进入触屏模式
		
		触摸屏幕左边坐标范围（0  ~250）查看上一张图片
		触摸屏幕右边坐标范围（550~800）查看下一张图片
		触摸中间屏幕坐标范围（250~550）回到等待命令模式
		
	6.指令 d  进入画图模式模式    
				屏幕自动清屏
				
				需要手动输入颜色和形状（同指令i）后，用手在屏幕上划一片区域，便可加上对应形状的色块。完成后继续等待输入颜色和形状，
				输入出错则退出画图模式，回到等待命令模式。
			
	7、指令 p  进入幻灯片播放模式
			
				幻灯片放映完自动退出到等待命令模式，（但中途无法暂停，有待优化）。
		
四、本项目代码基于C语言完成，在arm-linux平台下可使用，本项目仅适用于能够读取lcd屏幕压力值的系统。
